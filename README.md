# NER-for-Manufacturing-WI

This repository contains a named entity recognition (NER) corpus for manufacturing work instruction (WI).

## Contributer
- Peiyan Wang

## Overview
This corpus contains 709 fabricating WI sentences,  774 assembly WI sentences,  and total 427,656 characters. 

For the fabricating NER data, check the `Data/Fabricating/` folder.

For the assembly NER data, check the `Data/Assembly/` folder.

For total data, check the 'Data/Assembly&Fabricating/' folder.

And, fourteen types of entities are annotated. Following table shows the entities statistics.
